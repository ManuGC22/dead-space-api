Dead Space API

Overview

The objective of this API is for new or even experienced players of the Dead Space franchise, to know about all of the necromorphs (which are the in-game enemies) that they will encounter, what are their attacks and what strategy to use to defeat them.

Tutorial

Simply copy the url https://deadspace-api.cyclic.app/ and fetch it through your client-side Javascript.

To access the JSON data containing the information about a specific necromorph simply use the url and add the name of the necromorph as the last query parameter.

Each necromorph data contains their name, description, attacks and strategy to defeat them.

After you access the data through writing the name as the query parameter, you can access.

  • data['name'] to access the name of the necromorph.
  • data['description'] to access the description of the necromorph.
  • data['attack'] to access the attacks of the necromorph.
  • data['strategy'] to access the strategies to defeat the necromorphs.

Example

First fetch the url and add the last query parameter either with a constant value or a dynamic value. In this example we are fetching the slasher necromorph name.

Then we access the required information through the object name.

Lastly we can display that information in our website using the DOM.

let response = fetch(`https://deadspace-api.cyclic.app/api/slasher`) data = response.json() let necroName = data['name'] h1.innerText = necroName
